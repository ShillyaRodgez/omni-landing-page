# Omni-Data - Landing Page

Landing page moderna e responsiva para o projeto Omni-Data.

## 📋 Visão Geral

Esta landing page foi criada como sugestão de marketing para o projeto Omni-Data, um portal de relatórios e insights inteligentes baseado na integração com OMIE.

## 🚀 Funcionalidades

- **Design Responsivo**: Adaptável para desktop, tablet e mobile
- **Navegação Suave**: Scroll suave entre seções
- **Animações Modernas**: Efeitos de hover, transições e animações ao scroll
- **Formulário Interativo**: Validação de formulário com feedback visual
- **Sistema de Notificações**: Notificações customizáveis para feedback do usuário
- **Performance Otimizada**: Debounce de eventos e lazy loading
- **Acessibilidade**: Semântica HTML5 e navegação por teclado

## 🎨 Seções

1. **Header**: Navegação fixa com menu responsivo
2. **Hero**: Apresentação principal com dashboard preview animado
3. **Recursos**: Grid de funcionalidades com ícones e descrições
4. **Integração**: Detalhes da integração com OMIE
5. **Arquitetura**: Stack tecnológico utilizado
6. **Planos**: Tabela de preços com cards destacados
7. **Contato**: Formulário para solicitação de demonstração
8. **Footer**: Links e informações institucionais

## 🛠 Tecnologias Utilizadas

- **HTML5**: Semântica moderna e acessibilidade
- **CSS3**: Flexbox, Grid, animações e variáveis CSS
- **JavaScript Vanilla**: ES6+, Intersection Observer API
- **Font Awesome**: Ícones vetoriais
- **Google Fonts**: Tipografia Inter

## 🎨 Customização

### Cores
As cores estão definidas em variáveis CSS no início do arquivo `styles.css`:

```css
:root {
    --primary-color: #4f46e5;
    --secondary-color: #06b6d4;
    --accent-color: #f59e0b;
    /* ... */
}
```

### Fontes
A fonte Inter é carregada do Google Fonts. Para alterar, modifique o link no `<head>` do HTML.

### Animações
As animações usam CSS keyframes e podem ser ajustadas no arquivo `styles.css`.
